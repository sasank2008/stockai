# StockSense AI Dashboard

A glassmorphism stock forecasting dashboard with authentication, portfolio tracking, and simulated market data.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy the sample environment file and update values if needed:
   ```bash
   copy .env.example .env
   ```
3. Start the server:
   ```bash
   npm start
   ```

The app will be available at http://localhost:8080.
